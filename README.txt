Bienvenue dans ce "Jeu de devinette"

Ce repository contient un fichier html et un fichier js en plus du fichier README.txt
Pour fonctionner il faut créer un dossier html et y insérer le fichier html ensuite
il faut créer un dossier js et y insérer le fichier js. Ces 2 dossiers doivent être présent dans un même répertoire.

Ouvrir le fichier html avec firefox, une boite de dialogue apparaitra.

Puis toujours sur firefox faites outils, développement web et cliquez sur outils de développement pour ouvrir la console.

Positionnez-vous sur l'onglet console.

Et à vous de jouer.
